# Titanic Survival Analysis

## Project Overview
Exploratory Data Analysis and survival prediction on the Titanic dataset using Python.

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Analysis
4. Visualization
5. Survival Pattern Insights

## Objectives
- Perform data cleaning and preprocessing
- Explore survival patterns
- Visualize important features


## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn


## Dataset
Titanic dataset from Kaggle.

## Key Insights
- Female passengers had higher survival rate
- First-class passengers survived more frequently


## How to Run
1. Clone repository
2. Install dependencies:
   pip install -r requirements.txt
3. Open Jupyter Notebook and run cells
